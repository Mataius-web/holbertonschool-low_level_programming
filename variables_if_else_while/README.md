# C - Variables, if, else, while

> Mastering control flow, arithmetic operators, and the foundational building blocks of C logic.

## Description

This project is part of the **Holberton School**. It explores variables, data types, operators, and control flow structures in C — the essential tools for writing programs that make decisions and repeat actions.

## Learning Objectives

- Understand arithmetic, logical, relational, and boolean operators
- Use `if`, `if...else` statements to control program flow
- Use `while` loops to repeat instructions
- Declare and use variables of types `int`, `char`
- Understand the ASCII table and character encoding
- Apply the Betty coding style

## Requirements

- Ubuntu 20.04 LTS
- gcc (GNU Compiler Collection)
- All files compiled with: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows the **Betty** style guide
- No use of `printf`, `puts`, `strlen` — only `_putchar`

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.c` | Print whether a random number is positive, negative, or zero |
| `1-last_digit.c` | Print the last digit of a random number |
| `2-print_alphabet.c` | Print the lowercase alphabet |
| `3-print_alphabets.c` | Print lowercase and uppercase alphabets |
| `4-print_alphabt.c` | Print the alphabet excluding `e` and `q` |
| `5-print_numbers.c` | Print single-digit numbers |
| `6-print_numberz.c` | Print single-digit numbers using `_putchar` only |
| `7-print_tebahpla.c` | Print the alphabet in reverse |
| `8-print_base16.c` | Print all hexadecimal digits in lowercase |
| `9-print_comb.c` | Print all combinations of single-digit numbers |
| `10-print_comb2.c` | Print all combinations of two-digit numbers |
| `100-print_comb3.c` | Print all combinations of two different digits |
| `101-print_comb4.c` | Print all combinations of three different digits |
| `102-print_comb5.c` | Print all combinations of two two-digit numbers |

## Usage

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c _putchar.c -o output
./output
```
