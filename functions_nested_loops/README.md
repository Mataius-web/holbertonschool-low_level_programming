# C - Functions & Nested Loops

> Building reusable logic with functions and mastering nested iteration patterns.

## Description

This project is part of the **Holberton School**. It covers the creation and use of functions in C, the concept of scope, header files, and the power of nested loops to produce complex output patterns.

## Learning Objectives

- Understand the difference between a declaration and a definition
- Know what a prototype is and how to use it
- Understand variable scope (local vs global)
- Use nested loops to generate structured output
- Create and include custom header files
- Apply the Betty coding style consistently

## Requirements

- Ubuntu 20.04 LTS
- gcc (GNU Compiler Collection)
- All files compiled with: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows the **Betty** style guide
- No global variables
- No more than 5 functions per file
- Only `_putchar` allowed (no `printf`, `puts`)

## Files

| File | Description |
|------|-------------|
| `0-holberton.c` | Print a specific pattern to stdout |
| `1-alphabet.c` | Print the lowercase alphabet |
| `2-print_alphabet_x10.c` | Print the lowercase alphabet 10 times |
| `3-islower.c` | Check for lowercase character |
| `4-isalpha.c` | Check for alphabetic character |
| `5-sign.c` | Print the sign of a number |
| `6-abs.c` | Compute the absolute value of an integer |
| `7-print_last_digit.c` | Print the last digit of a number |
| `8-24_hours.c` | Print every minute of the day |
| `9-times_table.c` | Print the 9 times table |
| `10-add.c` | Add two integers and return the result |
| `11-print_to_98.c` | Print all natural numbers from `n` to 98 |
| `main.h` | Header file with all function prototypes |

## Usage

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c _putchar.c -o output
./output
```
