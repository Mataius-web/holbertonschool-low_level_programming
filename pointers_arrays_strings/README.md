# C - Pointers, Arrays and Strings

> Understanding memory addresses, pointer arithmetic, and string manipulation in C.

## Description

This project is part of the **Holberton School** curriculum. It covers pointers, arrays, and strings in C — how to manipulate memory directly, work with arrays using pointer notation, and implement standard string functions from scratch.

## Learning Objectives

- Understand what pointers are and how to use them
- Know the difference between pointers and arrays
- Understand how strings are stored in memory
- Use pointer arithmetic to traverse arrays and strings
- Implement string functions without the standard library

## Requirements

- Ubuntu 20.04 LTS
- gcc (GNU Compiler Collection)
- All files compiled with: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows the **Betty** style guide
- No global variables
- No more than 5 functions per file

## Files

| File | Description |
|------|-------------|
| `0-reset_to_98.c` | Set the value of an integer via a pointer to 98 |
| `0-strcat.c` | Concatenate two strings |
| `1-strncat.c` | Concatenate two strings using at most `n` bytes |
| `1-swap.c` | Swap the values of two integers |
| `100-atoi.c` | Convert a string to an integer |
| `2-strlen.c` | Return the length of a string |
| `2-strncpy.c` | Copy a string using at most `n` bytes |
| `3-puts.c` | Print a string followed by a newline |
| `3-strcmp.c` | Compare two strings |
| `4-print_rev.c` | Print a string in reverse |
| `4-rev_array.c` | Reverse the content of an array of integers |
| `5-rev_string.c` | Reverse a string in place |
| `5-string_toupper.c` | Change all lowercase letters to uppercase |
| `6-cap_string.c` | Capitalize all words in a string |
| `6-puts2.c` | Print every other character of a string |
| `7-leet.c` | Encode a string into leet speak |
| `7-puts_half.c` | Print the second half of a string |
| `8-print_array.c` | Print the elements of an array of integers |
| `9-strcpy.c` | Copy a string from source to destination |
| `main.h` | Header file with all function prototypes |

## Usage

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c -o output
```
