# C - Simple Calculator

> A basic interactive calculator built in C using `scanf` and conditional logic.

## Description

This project is part of the **Holberton School** curriculum. It implements a simple command-line calculator in C that presents a menu to the user and performs basic arithmetic operations based on the chosen option.

## Learning Objectives

- Use `scanf` to read user input
- Use `if/else` statements to handle menu choices
- Build a simple interactive program in C
- Apply the Betty coding style

## Requirements

- Ubuntu 20.04 LTS
- gcc (GNU Compiler Collection)
- All files compiled with: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows the **Betty** style guide

## Files

| File | Description |
|------|-------------|
| `calculator.c` | Main file — displays a menu and handles user input for basic operations |

## Usage

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator
./calculator
```

4) Divide
